Detailed Description

Concrete implementation of TouchScreenAdapter for Adafruit's FT6206 TouchScreen class.

Although the FT6206 reports X and Y as pixel position, both axis are reversed! (ie higher numbers are up or left). So display width and height are required in order to reverse them.
